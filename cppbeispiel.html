C++ Datenbank erklärt

In einem While Loop in der main Int, wird die Tasteneingabe interpretiert welches die Aktion festlegt. Es wird entweder die Liste herausgegeben, ein Eintrag festgelegt oder die Liste gelöscht. Die Entscheidung fällt eine If-Else-Else If Bedingung. Es werden nur Standart Bibliotheken verwendet inklusive fstream und string.
